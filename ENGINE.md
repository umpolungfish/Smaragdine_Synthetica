## Section Topology

The 15 versicles form a ⟨𐑑⟩ categorical chain — each versicle's output is the next versicle's input:

```
V01 → V02 → V03 → V04 → V05 → V06 → V07 → V08 → V09 → V10 → V11 → V12 → V13 → V14 → V15
```

The chain closes back to V01 — integer winding (𐑭) ensures the full cycle is one complete topological turn. The stone is grown, not synthesized.

## Transcription Format

Each versicle is transcribed as an 8-token IMASM sequence:

```
<SEAL> <ASCENT> <DESCENT> <SEPARATION> <UNION> <FIXATION> <ROTATION> <PROJECTION>
```

Tokens are rendered in Shavian IG primitives.
